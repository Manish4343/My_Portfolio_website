My_Portfolio_website
My Personal Portfolio is a fully responsive web application built to showcase my projects, technical skills, education, certifications, and professional experience. The website emphasizes clean UI/UX, smooth animations, and optimized performance to provide visitors with an engaging and seamless browsing experience across all devices.


Features

- Modern and responsive design
- Smooth scrolling and animations
- Interactive user interface
- Project showcase with live demo and GitHub links
- Skills and technology section
- Education and experience timeline
- Downloadable resume
- Contact form with social media links
- Mobile-friendly layout
- Optimized performance and accessibility
